<a id="Introdução"></a>
## 🧩 Introdução 

  ***⠀⠀⠀⠀Este é um projeto de estudo de criação de um leitor de áudio utilizando Next.js, Sass, HTML e JavaScript. O objetivo é demonstrar como essas tecnologias podem ser utilizadas em conjunto para criar uma solução elegante e funcional. Sinta-se à vontade para verificar o código fonte e experimentar o leitor de áudio!.***

```BASH

```

<br /> 

> Caso já tenha o Node em sua maquina basta instalar o projeto com npm i

